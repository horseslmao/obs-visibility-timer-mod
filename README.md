# obs-visibility-timer-mod

This is a visibility timer script for OBS Studio with the following options:

1. Source: select source
2. Scene: select scene
3. Mode: Show source after specified time, Hide source after specified time, Repeat
4. Delay after activated (ms): delay timer until after specified time (useful if using scene transition)
5. Duration (ms): time the source will hide or show
6. Repeat hold time (ms): repeat mode only - time until which the source will hide or show again
7. Start visible (repeat mode): starts the source shown or hidden in repeat mode
